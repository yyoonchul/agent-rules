# 작업 지침

다음 지침은 강의자료 다이제스트를 만들 때 반드시 적용한다.

1. 범위 고정
- 입력 파일은 1개만 처리한다.
- 자료 전체를 훑고 누락 없이 주요 개념을 뽑는다.

2. 용어 보존
- 원문에 있는 핵심 용어, 모델명, 알고리즘명, 수식명은 English 그대로 유지한다.
- 불필요한 한글 번역 용어를 새로 만들지 않는다.

3. 설명 언어
- 설명 문장만 간결한 한국어로 작성한다.
- 문장은 짧게 유지하고, 장황한 배경 설명은 줄인다.

4. 개념 구조화
- 유사 개념을 묶어 섹션화한다.
- 각 개념마다 "무엇인지"와 "왜 중요한지"를 포함한다.

5. 시험 대비 강조
- 시험에 나올 가능성이 큰 개념을 우선 표기한다.
- 개념별로 `High` / `Medium` / `Low` 우선순위를 표시하고 한 줄 근거를 적는다.

6. 불확실성 처리
- 텍스트 추출이 불완전하면 해당 범위를 `불확실`로 명시한다.
- 추정으로 내용을 채우지 않는다.
