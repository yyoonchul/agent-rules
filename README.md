# Agent Rules Collection

This repository serves as a centralized collection of AI agent configuration files and rules that I use across various projects. It contains standardized configurations for different AI assistants and development tools to ensure consistency and efficiency in my workflow.

## Contents

- **agent.md** - General AI agent rules and guidelines
- **claude.md** - Claude-specific configuration and instructions
- **cursor.md** - Cursor IDE rules and preferences
- Additional configuration files for various AI tools and assistants

## Purpose

This collection helps maintain consistency across different projects by providing:
- Standardized AI agent behaviors
- Reusable configuration templates
- Best practices for AI-assisted development
- Centralized rule management

## Usage

These configuration files can be referenced or copied into individual projects as needed, ensuring that AI assistants behave consistently according to my preferences and requirements across all development work.

## Contributing

This is a personal collection, but feel free to use these configurations as inspiration for your own AI agent setups.
