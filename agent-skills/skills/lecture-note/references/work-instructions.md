# 작업 지침

다음 지침은 전사본에서 강의노트를 생성할 때 반드시 적용한다.

1. 노이즈 필터링
- 음성 인식 과정의 오탈자, 불필요한 추임새(예: 어, 음, 그니까), 문맥상 끊김을 자연스럽게 교정한다.
- 단, 과교정 금지: 의미가 바뀔 수 있는 적극적 재작성은 피하고 원문 보존을 우선한다.
- `inaudible`류 표기나 반복 문자 노이즈는 느슨하게 정리한다.

2. 핵심 요약
- 강의 핵심 주제를 파악해 1~2문장의 TL;DR을 작성한다.

3. 목차 추출
- 강의 흐름에서 대주제를 추출해 목차를 순서대로 작성한다.

4. 논리적 구조화
- 시간순 나열보다 주제 중심 구조를 우선한다.
- 대주제별 하위 내용을 가독성 높은 불릿으로 정리한다.

5. 용어/수식 시각화
- 중요 전공 용어, 영문 키워드, 공식/법칙은 `**굵게**`, 인라인 코드, 코드 블록 등을 활용해 명확히 표현한다.

6. 강조 포인트 보존
- 교수의 반복 강조, 과제 안내, 시험 힌트는 누락하지 말고 별도 섹션에 정리한다.
